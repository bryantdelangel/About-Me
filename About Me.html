Production Website is down? Now what…

If a production website is down the first order of business is to visit the website from a separate device and make sure it is not an isolated issue with the user’s device.

We need to ensure what the main issue is.

We ping the domain to make sure the server is up and reachable, we can find that either the server crashed and/or programming error. If not successful, we would run a traceroute to pinpoint where the issue could be.

We also need to verify the domain has not expired. (Who.is a great tool to find out the status of the domain)

If the server has crashed completely we will need to fire up the server and figure out what happened.
